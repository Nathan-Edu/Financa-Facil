# Requisitos Funcionais

| Código | Requisito Funcional |
|---|---|
| RF01 | O sistema permitirá o cadastro de usuários. |
| RF02 | O sistema permitirá o login de usuários cadastrados. |
| RF03 | O sistema permitirá o cadastro de perfis financeiros pessoais e empresariais. |
| RF04 | O sistema permitirá o cadastro de receitas. |
| RF05 | O sistema permitirá o cadastro de despesas. |
| RF06 | O sistema permitirá o gerenciamento de categorias financeiras. |
| RF07 | O sistema permitirá o cadastro e controle de contas parceladas. |
| RF08 | O sistema permitirá registrar o pagamento de parcelas. |
| RF09 | O sistema calculará automaticamente o saldo financeiro de cada perfil cadastrado. |
| RF10 | O sistema permitirá a consulta do histórico de transações com filtros por categoria, período e tipo. |
| RF11 | O sistema permitirá a geração de relatórios financeiros por período e perfil financeiro. |
| RF12 | O sistema emitirá alertas de vencimento de parcelas. |
| RF13 | O administrador poderá gerenciar usuários cadastrados. |
| RF14 | O sistema validará o CPF para perfis pessoais e o CNPJ para perfis empresariais no momento do cadastro. |
| RF15 | O sistema bloqueará o acesso do usuário após 5 tentativas de login incorretas. |
| RF16 | O sistema permitirá a alteração de senha pelo usuário. |
| RF17 | O sistema permitirá a atualização da renda do usuário para refletir reajustes ou recebimentos extras. |
| RF18 | O sistema permitirá a recuperação de senha por e-mail. |
| RF19 | O sistema permitirá a edição e exclusão de receitas e despesas cadastradas. |
| RF20 | O sistema permitirá o cancelamento de uma conta parcelada. |
| RF21 | O sistema atualizará automaticamente o status das parcelas vencidas não pagas. |
