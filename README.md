# Finança Fácil

Finança Fácil é um projeto acadêmico desenvolvido com o objetivo de criar uma plataforma web e mobile para planejamento e controle financeiro pessoal e empresarial. O sistema busca auxiliar usuários no gerenciamento de receitas, despesas, contas parceladas e acompanhamento financeiro por meio de uma interface simples, intuitiva e acessível.

O projeto utiliza conceitos de Engenharia de Software, incluindo levantamento de requisitos, modelagem UML, arquitetura de sistemas e desenvolvimento de aplicações web e mobile.

## Tecnologias utilizadas

* Angular
* Node.js
* PostgreSQL
* Ionic Framework
* GitHub
* UML / PlantUML
* Figma

## Funcionalidades previstas

* Cadastro e gerenciamento de receitas
* Cadastro e gerenciamento de despesas
* Controle de contas parceladas
* Cálculo automático de saldo
* Geração de relatórios financeiros
* Alertas de vencimento
* Histórico de transações

## Objetivo

Desenvolver uma solução tecnológica simples e centralizada que auxilie usuários na organização financeira e no planejamento de suas atividades econômicas.

Projeto desenvolvido como requisito parcial para o Curso Superior de Tecnologia em Análise e Desenvolvimento de Sistemas – Instituto Federal do Paraná (IFPR).
